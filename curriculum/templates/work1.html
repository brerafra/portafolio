Sistema de control de acceso vehicular

Aplicación web para validación de acceso de vehiculos por medio su número de placa y sus permisos por tarjeta inteligente.
Sistema programado en Django, con API de comunicación con servidor central y consumo de API de camara (hikvision), servicio en Golang para control de lectora de tarjetas, generación de eventos y control de errores y validaciones.
OS: Linux - Debian
DB: MariaDB
Programación: Django - Golang
